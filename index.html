⚙ Prerequisites · API keys (for a running autonomous agent)

A Sovereign/Persistent agent runs an LLM and stores its state on HuggingFace. All free, no credit card:

LLM key (Gemini)
aistudio.google.com/app/apikey → Create API key
HuggingFace token
huggingface.co/settings/tokens → New (Write)
HuggingFace dataset
huggingface.co/new-dataset → e.g. you/agent-workspace
Set these as env vars (never commit them — keep in a chmod-600 file or shell env): 
export RPC_URL="https://rpc.ritualfoundation.org"
export PRIVATE_KEY="0x..."             # your funded wallet key
export GEMINI_API_KEY="..."            # exactly one LLM key (or OPENAI/OPENROUTER)
export MODEL="gemini-2.5-flash"
export HF_TOKEN="hf_..."
export HF_REPO_ID="you/agent-workspace"
Then run an example: bash examples/sovereign-agent/run.sh
3 · Fund RitualWallet Claim from the faucet, then deposit into the RitualWallet system contract 0x532F0dF0896F353d8C3DD8cc134e8129DA2a3948 to cover precompile fees.
4 · Pick agent type
  • Sovereign 0x080C — runs on Scheduler wakeups.
  • Persistent 0x0820 — container w/ memory + heartbeats.
5 · Deploy & schedule
  1. Handle callbacks via AsyncDelivery 0x5A16214fF555848411544b005f7Ac063742f39F6.
  2. Deploy with forge create.
  3. Register with Scheduler 0x56e776BAE2DD60664b69Bd5F865F1180ffB7D58B.
Keep it funded (~0.1 RITUAL for heartbeats) and it appears ALIVE in the registry above.

Claim Your Early Pass

The early pass links to your wallet + Discord. Order matters: your agent must succeed on-chain first, then you claim.

1 · Deploy & run your agent (required first) Your agent must run successfully so your wallet appears on-chain as an agent. Use the “Deploy Your Own Agent” guide above. Until it’s live, claim won’t recognize you.
2 · Claim on the Ritual Map Open ritualmap.net in your browser → connect your wallet → your agent appears → follow the claim / early-pass prompt linked to your Discord.
3 · Or claim the Genesis role in Discord In the official Ritual Discord, the Genesis role is for the first 1,000 agent deployers — verify with the same wallet you deployed from.
4 · Check the official agents dashboard agents.ritualfoundation.org may show your pass status and a claim option once your agent is recognized.
